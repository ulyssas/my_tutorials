[MMD Tools Append](https://github.com/MMD-Blender/blender_mmd_tools_append) (formerly known as MMD UuuNyaa Tools) is a Blender addon that works with MMD Tools to add features that focuses on Blender workflows.

## Rigify Integration

{% include youtubePlayer.html id="av9rYRu0F8c" %}

---

With Rigify Integration, you can control MMD models using Rigify rigs. You can import VMD files directly to integrated rigs.

## Physics Adjuster

{% include youtubePlayer.html id="pRJNJDFSYfk" %}

---

You can convert MMD Rigid body physics to cloth simulation.

## Material Assets

{% include youtubePlayer.html id="UkefaCNpmeU" %}

---

MMD Tools Append includes some material assets to make models instantly look better.

[List of available materials](https://github.com/MMD-Blender/blender_mmd_tools_append/wiki#setup-materials)

## Humanoid Renamer

{% include discord-gallery.html id="gallery" %}

Humanoid Renamer allows you to assign MMD bone names to models automatically. It can also convert models into MMD in one click.

You can read more about it from [here]({{ site.baseurl }}{% post_url 2026-06-03-014-fbx-to-pmx %}#convert-directly-mmd-tools-append).
