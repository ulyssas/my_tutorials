| Blender LTS Version | MMD Tools Version | MMD UuuNyaa Tools Version |
| ------------------- | ----------------- | ------------------------- |
| Blender 4.2 & 4.5   | MMD Tools v4.x    | MMD Tools Append v4.x     |
| Blender 3.6         | MMD Tools v2.10.3 | UuuNyaa Tools v1.8.0      |
| Blender 2.93        | MMD Tools v2.10.3 | UuuNyaa Tools v1.7.2      |
| Blender 2.83        | MMD Tools v2.5.1  | UuuNyaa Tools v1.6.0      |

---

If you want to use MMD Tools Append on Blender 4.0 & 4.1, install [MMD Tools Addon](https://github.com/ulyssas/mmd_tools_addon_4.2/releases/) and [patched UuuNyaa Tools](https://github.com/ulyssas/mmd_uuunyaa_tools_4.2/releases/tag/v1.8.0_patch).

Some features are missing in this patched UuuNyaa Tools. If you need them, you can create issues in the repository.
