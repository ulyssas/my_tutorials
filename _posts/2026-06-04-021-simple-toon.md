It's a Goo Engine tutorial, but first one works for standard blender as well.

{% include discord-gallery.html id="gallery1" %}

Emission is the key here. It eliminates soft, realistic shadow. And Shader to RGB node enables NPR style shader.

Area lights in the scene is for lighting up her face. Without them, she will have dark face.

{% include discord-gallery.html id="gallery2" %}

In this scene, View Transform in Render properties is set to `Standard` so that it respects texture color.

{% include discord-gallery.html id="gallery3" %}

She has transparent texture for eyelashes, which causes them to turn black.

This node setup adds transparency cutout to fix that.

{% include discord-gallery.html id="gallery4" %}
