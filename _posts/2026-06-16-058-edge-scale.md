Some models look weird because edges become visible when it shouldn't.

{% include discord-gallery.html id="gallery1" %}

You can remove edges from specific vertices with `Edge Scale`. Select the vertices you want to remove edges from, then go to `Edit > Vertex > Edge Scale > 0`.

{% include discord-gallery.html id="gallery2" %}

Now the model should look better!
